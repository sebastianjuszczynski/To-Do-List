# To-Do List (Vanilla JavaScript) 📝

A simple task management application built with **HTML, CSS and Vanilla JavaScript**.

This project was created to practice fundamental frontend development concepts such as DOM manipulation, event handling, and dynamic rendering of content.

---

## Live Demo

👉 https://sebastianjuszczynski.github.io/To-Do-List/

---


# Project Overview

The goal of this project was to improve core JavaScript skills by building a small but practical web application.

While developing this project I focused on:

- manipulating the DOM
- handling user input
- dynamically creating and removing elements
- working with events in JavaScript
- structuring a small frontend application

---

# Features

- Add new tasks
- Remove tasks
- Mark tasks as completed
- Dynamic task list rendering
- Simple and clean user interface

# Installation

Clone the repository

```bash
git clone https://github.com/sebastianjuszczynski/To-Do-List.git
```

Navigate to the project directory

```bash
cd To-Do-List
```

Open the project

Simply open the `index.html` file in your browser.
